# Stock Data Extraction and Visualization

## Objective
Extract historical stock data and visualize revenue and stock price trends.

## Dataset
- Stock price data using yfinance API
- Revenue data extracted from web sources

## Techniques Used
- Web scraping using BeautifulSoup
- Data extraction using yfinance
- Data cleaning with Pandas
- Data visualization using Matplotlib

## Key Insights
- Compared stock price trends with company revenue
- Identified periods of growth and decline

## Tools
Python, Pandas, Matplotlib, yfinance, BeautifulSoup
